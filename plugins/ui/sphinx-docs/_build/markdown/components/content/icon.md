### deephaven.ui.icon(name: str, \*children, \*\*props)

Based on Spectrum’s [Icon](https://react-spectrum.adobe.com/react-spectrum/Icon.html) component.
Get an icon by name.

Usage:

```python
my_icon = ui.icon("dhTruck")
```

* **Parameters:**
  **name** – The name of the icon to render.
* **Returns:**
  The icon element.
